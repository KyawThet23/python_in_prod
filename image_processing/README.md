## What is a Digital Image ?
- A digital image can be represented as a rectangular array of numbers.
- Images are comprised with rectangular grid of Pixels.
- Each pixel has therir own color value and intensity value.
- Color value represents how much red, green and blue.
- Intensity value represents pixel brightness.
- They both in the range of 0-256 becaue of 8-bit quantization.
- Computer only understands image in the form of number matrix which has same number of elements in each row.
- The most used libraries are openCV and pillow
